# Rajasthan Tour Website - Frontend

React-based frontend for the Rajasthan Tour Website.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app runs on `http://localhost:3000`

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/        # Reusable components
│   ├── Navbar.tsx
```
